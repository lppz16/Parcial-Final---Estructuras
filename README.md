Sistema de Gestión de Estudiantes

Este software, desarrollado en C++, permite administrar la información académica básica de estudiantes, facilitando tareas como el registro, búsqueda, cálculo de promedios, eliminación y análisis estadístico de datos. Fue creado con fines académicos en el contexto de la materia Estructura de datos y algoritmos.

El programa inicia cargando automáticamente una lista de estudiantes de prueba con sus respectivas notas y muestra al estudiante con mejor desempeño. A partir de ahí, el usuario puede interactuar con un menú que ofrece distintas opciones, entre ellas agregar un estudiante de forma manual, buscarlo por su ID, visualizar al mejor promedio, mostrar los tres mejores estudiantes, listar todos los registros, ver estadísticas generales o eliminar un estudiante.

Internamente, el sistema se organiza en dos partes principales: la estructura “Estudiante” y la clase “SistemaEstudiantes”. La primera representa a cada estudiante, con sus datos personales, sus notas y su promedio, validando que la información sea correcta y calculando automáticamente el promedio de calificaciones. La segunda se encarga de administrar el conjunto de estudiantes mediante un registro que permite realizar las operaciones de búsqueda, ordenamiento, eliminación y generación de estadísticas.

El programa utiliza conceptos de programación orientada a objetos, manejo de excepciones, estructuras de datos como vectores y mapas, y operaciones de ordenamiento para determinar los mejores desempeños. Está diseñado para ejecutarse en consola y mantener una interacción clara y sencilla con el usuario.

En resumen, este sistema constituye una herramienta práctica y didáctica para gestionar información estudiantil, aplicando los fundamentos de la programación estructurada y orientada a objetos en C++.
